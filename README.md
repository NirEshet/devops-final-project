# DevOps Final Project## OverviewThis project demonstrates a complete DevOps deployment of the NameGen application on AWS EKS.The application is containerized with Docker, stored in AWS ECR, deployed on Kubernetes (EKS), exposed through a Network Load Balancer (NLB), and connected to a MongoDB database running as a StatefulSet with persistent storage.## ArchitectureGitHub Repository↓GitHub Actions↓AWS ECR↓AWS EKS↓NameGen Application↓MongoDB StatefulSet↓Persistent Volume## Technologies- AWS EKS- AWS ECR- Kubernetes- Docker- GitHub Actions- MongoDB- Node.js- eksctl## Project Structure```text.github/workflows/     GitHub Actions CI/CD pipelinediagrams/              Architecture diagrameks/                   EKS cluster configurationk8s/                   Kubernetes manifestsnamegen/               Application source codescreenshots/           Running application screenshotsREADME.md              Project documentation```## Kubernetes Components### Application- NameGen Deployment- NameGen Service (LoadBalancer / NLB)### Database- MongoDB StatefulSet- MongoDB Service- Persistent Volume- Persistent Volume Claim## CI/CD PipelineEvery push to the main branch automatically:1. Builds a Docker image2. Pushes the image to AWS ECR3. Connects to AWS EKS4. Updates the deploymentThe workflow configuration is located in:```text.github/workflows/deploy.yml```## Deployment Files### EKS```texteks/cluster.yaml```### Kubernetes```textk8s/app-deployment.yamlk8s/app-service.yamlk8s/mongodb-service.yamlk8s/mongodb-statefulset.yamlk8s/mongodb-pv.yamlk8s/mongodb-pvc.yamlk8s/gp3-storageclass.yaml```## Verification CommandsCheck Pods:```bashkubectl get pods```Check Services:```bashkubectl get svc```Check Deployments:```bashkubectl get deployments```Check StatefulSets:```bashkubectl get statefulset```Check Persistent Volumes:```bashkubectl get pv```## ScreenshotsThe screenshots folder contains evidence of:- GitHub Repository- GitHub Actions Pipeline- ECR Repository- EKS Cluster- Deployments- Pods- Services- StatefulSet- Persistent Volume- Running ApplicationNir Eshet
